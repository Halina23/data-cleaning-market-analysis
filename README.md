# 📊 Data Cleaning & Market Analysis (Python + Pandas)

## 🎯 Objetivo

Este projeto tem como objetivo realizar o tratamento, organização e análise de dados a partir de planilhas Excel com múltiplas abas, visando gerar uma base estruturada para análise e tomada de decisão.

---

## 🛠️ Tecnologias Utilizadas

- Python  
- Pandas  
- OpenPyXL  
- Excel  

---

## 🔄 Etapas do Projeto

### 1. Leitura dos Dados

- Importação de arquivos Excel com múltiplas abas  
- Identificação da estrutura dos dados  

### 2. Tratamento

- Limpeza de dados (valores nulos, inconsistências)  
- Padronização de colunas  
- Organização das informações  

### 3. Organização

- Separação entre dados brutos (`data/`) e dados tratados (`outputs/`)  
- Estruturação para facilitar análise  

### 4. Geração de Saída

- Exportação dos dados tratados para arquivos CSV  
- Armazenamento na pasta `outputs/`
